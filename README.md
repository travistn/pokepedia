# PokéPedia
A database to find all things about Pokémon.

### [Live Demo](https://pokepedia-tn.netlify.app/)

## Description
This app consumes data from [PokéAPI](https://pokeapi.co/) to build a UI that displays the API's data.

## Technologies Used
* React.js
* Javascript
* CSS
* HTML
* React Router
* Tailwind CSS
* Redux Toolkit Query
